# Google Search Maps Button

A browser extension that brings back the Google Maps button to Google Search.

## Features

- ✅ Automatically adds a "Maps" button to Google search results
- ✅ Works with all Google domains (google.com, google.de, google.at, google.ch)
- ✅ Opens Google Maps with the current search term
- ✅ Google-like design and styling

## Development

```bash
# Install dependencies
pnpm install
# Start development server (Chrome)
pnpm dev
# Build for production (Chrome)
pnpm build
```

## Supported websites

- google.com
- google.de
- google.at
- google.ch

## Technical details

- **Framework:** WXT (Web Extension Toolkit)
- **Language:** TypeScript
- **Manifest:** V3 (Chrome)
- **Content Scripts:** Runs on Google search pages
- **Permissions:** Only host permissions for Google domains
